# AWS 3-Tier Architecture

## Project Overview
Designed and implemented a highly available 3-Tier Architecture on AWS.

## AWS Services Used
- VPC
- EC2
- RDS
- IAM
- Elastic Load Balancer (ELB)
- Auto Scaling
- CloudWatch

## Features
- Public and Private Subnets
- Secure Network Architecture
- Auto Scaling
- Load Balancing
- Database Integration
- Monitoring and Alerts
